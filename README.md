# Ejercicios de Programación con Interfaz Gráfica
### Python · Tkinter

Proyecto académico con 10 ejercicios interactivos desarrollados con interfaz gráfica en Python usando la biblioteca **Tkinter**.

---

## Cómo ejecutar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/TU_REPO.git
   cd TU_REPO
   ```

2. Asegúrate de tener Python 3 instalado (Tkinter viene incluido).

3. Ejecuta el menú principal:
   ```bash
   python main.py
   ```
